## 👋 About Me

<img align="right" src="https://raw.githubusercontent.com/3n3a/3n3a/main/wave.gif" height="45" />

Hi! I'm **Arpit Tagade** — a first-year B.Tech (AI & Data Science) student building full-stack and backend systems at a level most people don't attempt before an internship. I'm the builder behind **[Monika-AI](https://github.com/tagadearpit/Monika-AI)**, **[Neosis](https://github.com/tagadearpit/Neosis)**, and **CandyRobot**.

A few things that are true about how I work:

- 🏗️ I build full-stack, end to end — Next.js/React on the frontend, Spring Boot and Node.js on the backend
- 🔐 I care about the unglamorous parts — authentication, security hardening, and real-time infrastructure (WebSockets, WebRTC)
- 🤖 I integrate AI into real products — vision, voice, and memory, not just a chatbot demo
- 🔧 I also build at the hardware layer — ESP32 firmware talking to servos, displays, and cloud AI
- 📚 Still in my first year — building like the deadline is an internship, not a syllabus
